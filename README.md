## SimilACTrail_v1

<img src="https://github.com/Amincheminfom/Amincheminfom/blob/main/Cheminform_logo.jpg" alt="pDILI Logo" width="350" align="right"/>

SimilACTrail_v1 is a Python package for visualizing the Structure-Similarity Activity Trailing map.

SimilACTrail_v1 stands for Structure-**SIMILA**rity **ACT**ivity **TRAIL**ing Map.

It also identifies **Activity Cliffs (AC)** in the dataset. 

It is a <img src="https://streamlit.io/images/brand/streamlit-mark-color.svg" alt="Streamlit Logo" width="50"/>-based [Web Application](https://similactrailv1web.streamlit.app/).

Just upload a CSV file with alteast three columns - (i) Molecule ID, (ii) SMILES and (iii) Activity column (e.g., pIC50) to explore SimilACTrail Map!
